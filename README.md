## 📋 About HemoConnect

HemoConnect is an easy-to-use forum designed to help people with hemophilia connect with others. It uses AI technology to match users with peers for support and conversation. This app works like a social platform where you can chat in real-time and join a friendly community. It runs on Windows and requires no technical knowledge to use.

## 💻 What You Need Before Starting

Before you download and run HemoConnect, make sure your computer meets these requirements:

- Operating System: Windows 10 or later  
- Internet connection: Required for logging in and chatting  
- Free Disk Space: At least 200 MB available

You do not need any special software or accounts before you start. HemoConnect will guide you through any setup steps.

## 🧰 Technical Details (Optional)

HemoConnect uses several technologies to provide a smooth experience:

- It runs on React and Next.js for a fast interface.  
- Real-time chat is powered by Supabase and PostgreSQL.  
- AI peer matching uses models hosted on Hugging Face.  
- The design uses Tailwind CSS for a clean look.  
- The app is deployed on Vercel for performance.  

These tools work behind the scenes so you don’t have to worry about anything technical.
